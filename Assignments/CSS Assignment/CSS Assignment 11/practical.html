About Pseudo-Elements

This paragraph demonstrates the use of the ::first-line pseudo-element. The first line, regardless of how wide the browser window is, will have a unique font size and color. This styling will dynamically adjust if you resize the window, proving it's styling the logical line break, not a fixed amount of text.
